Clínica Oftalmológica VistaClear

Um site completo e responsivo para uma clínica oftalmológica, desenvolvido com HTML e CSS puro.


📋 Descrição
A Clínica Oftalmológica VistaClear é um site profissional que apresenta informações sobre uma clínica especializada em oftalmologia. O projeto foi desenvolvido como desafio educacional, aplicando conceitos de HTML semântico, formulários, tabelas, mídias e estruturação de texto.



🎯 Funcionalidades
Página Principal: Apresentação da clínica e seus serviços
Sobre a Clínica: História, missão, visão, valores e informações sobre a equipe
Horário de Atendimento: Tabela com serviços e horários de funcionamento
Contato: Informações de contato, localização no Google Maps e formulário de contato


🏗️ Estrutura do Projeto
clinica-oftalmologica/
├── index.html          # Página principal
├── sobre.html          # Página sobre a clínica
├── horario.html        # Página de horários
├── contato.html        # Página de contato
└── README.md           # Este arquivo


🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica das páginas
CSS3: Estilização e responsividade
Google Maps API: Mapa interativo de localização
